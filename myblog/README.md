# 藍鯨 *文字博客網站*
## 操作指令
1. 啟動虛擬環境 `env\Scripts\activate.bat`
3. 安裝Django `py -m pip install Django`
4. 安裝Markdown `py -m pip install markdown`
5. 切換至專案目錄 `cd ~/myblog`
6. 啟動開發伺服器 `python manage.py runserver`
